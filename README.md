# Gravity UV (v5.0)

**Gravity UV** — это плагин для Blender 5.1.0+, разработанный для быстрого автоматического выравнивания текстурных UV-координат по направлению силы тяжести (мировому вектору вниз). 

Плагин решает частую проблему в геймдеве и архитектурной визуализации, когда текстуры (например, кирпичная кладка, потеки грязи, доски или бетон) накладываются на изогнутые или наклонные 3D-модели неровно. Gravity UV автоматически ориентирует текстуры строго вертикально.

---

## 🚀 Основные возможности

### 1. Подготовка развертки (Batch Unwrap)
* Позволяет быстро сделать стандартную развертку (Unwrap) с отступом (margin) 0.02 для всех выделенных 3D-объектов одной кнопкой.
* Автоматически фильтрует выделение: игнорирует источники света, камеры и пустышки, работая только с мешами (Mesh).

### 2. Выравнивание по гравитации (Gravity Align)
* **Создание 2-го UV-канала**: Плагин не портит вашу оригинальную развертку. Он копирует её структуру в новый отдельный UV-канал с именем `Gravity_UV`.
* **Умная группировка плоскостей**: Грани модели группируются по сонаправленности их нормалей. С помощью ползунка **«Искривление плоскости»** можно задать угол погрешности (в градусах). Это позволяет объединять изогнутые стены в единые островки для совместного поворота без образования швов и «зигзагов» на стыках полигонов.
* **Поворот по гравитации**: Для всех наклонных и вертикальных граней плагин высчитывает направление силы тяжести в 3D-пространстве и поворачивает UV-координаты так, чтобы низ текстуры смотрел строго вниз.
* **Упаковка с сохранением масштаба**: После поворота UV-островки автоматически распределяются по UV-сетке, чтобы они не перекрывали друг друга. При этом плагин **на 100% сохраняет оригинальный масштаб и Texel Density** (детали не растягиваются и не уменьшаются).

### 3. Обработка горизонтальных граней (Крыши и полы)
* Горизонтальные плоскости не имеют явного направления «вниз», поэтому для них предусмотрена отдельная логика.
* **Погрешность горизонтали**: Ползунок с высокой точностью (шаг **0.01 градуса**) позволяет настроить угол отклонения от вертикали, при котором грань начинает считаться плоской (горизонтальной).
* **Режимы обработки**: 
  * *Не учитывать (Ignore)* — горизонтальные полигоны остаются без изменений.
  * *Повернуть на угол (Align)* — поворачивает все плоские грани на заданный вручную угол.

### 4. Интерактивный предпросмотр
* Плагин создает временный технический материал предпросмотра, который автоматически возвращает оригинальный материал при отключении превью.
* **Тестовые стрелки**: Проецирует текстуру стрелок (`arrows.png`) на модель, чтобы наглядно показать направление гравитации на поверхностях.
* **Разметка плоских граней**: Визуально подсвечивает плоские горизонтальные поверхности предупреждающими полосами (`flat_warning.png`), сигнализируя, что здесь гравитация не работает. Подсветка динамически меняется прямо во вьюпорте при перемещении ползунка погрешности горизонтали.

## 🛠️ Интерфейс панели (N-панель)

Панель плагина располагается в правой боковой панели 3D Viewport во вкладке **Gravity UV** и разделена на 5 блоков:

1. **1. Подготовка**: Кнопка «Пересоздать развертку» для быстрой массовой подготовки мешей.
2. **2. Выравнивание**:
   * Кнопка «Выровнять по Гравитации».
   * Слайдер «Искривление плоскости» (погрешность группировки нормалей).
   * Галочка «Показывать карту Gravity_UV» (быстрое переключение отображения между оригинальным UV-каналом и выровненным Gravity_UV).
   * Галочка «Упаковать без масштабирования» (автоматическое распределение островков без перекрытий с сохранением плотности пикселей).
3. **3. Управление текстурами**:
   * Галочка «Включить тестовые стрелки» (превью направления).
   * Слайдер «Масштаб (тайлинг)» для быстрой настройки частоты повторения текстуры на объектах.
4. **4. Настройки плоских граней**:
   * Галочка «Разметка плоских граней» (включение предупреждающей текстуры-зебры для крыш/полов).
   * Слайдер «Погрешность горизонтали» (задание точного угла определения плоских граней).
   * Меню выбора режима обработки и слайдер угла поворота плоских граней.
5. **5. Настройки меша**:
   * Галочка «Собственное искривление» — позволяет переопределить глобальные настройки погрешности искривления для конкретного объекта.
   * Слайдер «Искривление плоскости» — индивидуальная погрешность для данного меша (отображается только при включенной галочке).

## 📦 Модульная архитектура SOLID (для разработчиков)

Плагин полностью переписан по SOLID принципам и разбит на отдельные файлы для легкого расширения и поддержки:
* `__init__.py` — Точка входа. Содержит метаданные и регистрирует/удаляет плагин в Blender.
* `core.py` — Чистая математика и геометрия. Содержит логику поворота UV по гравитации и алгоритм бесконфликтной упаковки островков.
* `properties.py` — Хранит переменные сцены/объектов и управляет триггерами обновления вьюпорта.
* `material.py` — Генерирует нодовое дерево для превью-материала во вьюпорте.
* `ui.py` — Отвечает только за отрисовку кнопок и меню на панели.
* `operators.py` — Связывает нажатия кнопок в интерфейсе с вызовом логики из `core.py`.
